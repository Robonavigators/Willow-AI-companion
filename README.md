# Willow: AI Companion

![Status: Active](https://img.shields.io/badge/Status-Active-brightgreen.svg?style=for-the-badge)
![Hardware: XIAO ESP32-S3](https://img.shields.io/badge/Hardware-XIAO_ESP32--S3-blue.svg?style=for-the-badge)

Willow is an interactive AI companion project powered by the **Seeed Studio XIAO ESP32-S3**. Designed as a high-fidelity expressive desktop companion, Willow features a crisp **0.96-inch SSD1306 OLED (Full White)** display, clear audio output via the **MAX98357A I2S Amplifier**, and precision voice capture using the **INMP441 MEMS Omnidirectional Microphone**.

---

## Features
*   **Expressive Interface:** Dynamic facial animations on a high-contrast white OLED display.
*   **Audio Capability:** Powered by a 3W 4Ω speaker for clear, resonant AI vocal feedback.
*   **Voice Recognition:** High-precision audio capture using the professional-grade INMP441 MEMS microphone.
*   **Compact & Modular:** Optimized for the powerful ESP32-S3 architecture, ideal for custom enclosures.

---

## Hardware Requirements
*   **Microcontroller:** Seeed Studio XIAO ESP32-S3.
*   **Display:** 0.96" SSD1306 OLED (Full White).
*   **Audio Output:** MAX98357A I2S Amplifier + 4Ω 3W Speaker.
*   **Audio Input:** INMP441 MEMS Omnidirectional Microphone.

---

## Installation (Quick Flash)

To install the firmware without setting up the full development environment, follow these steps:

1. Navigate to the  [Robonavigators ESP Flasher](https://robonavigators.github.io/flash.html)
2. Select "Willow: AI companion" firmware from the menu.
3. Connect your XIAO ESP32-S3 via USB.
4. Select your COM/Serial port in the browser interface.
5. Click "Upload" to flash the firmware.
